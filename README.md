# 🌿 Paradise Nursery - E-Plant Shopping App

An elegant shopping cart application for an online plant shop offering a variety of house plants.

## Features

- 🏠 Landing page with hero section and shop button
- 🧭 Navigation bar with links to all pages
- 🌱 Product listing with two categories: Aromatic Plants & Medicinal Plants
- 🛒 Shopping cart with quantity management
- ➕/➖ Adjust quantities or delete items
- 💳 Checkout and continue shopping functionality
- 📱 Fully responsive design

## Technologies Used

- React 18
- Redux Toolkit (State Management)
- React Router DOM v6
- CSS3 with modern flexbox/grid
- GitHub Pages for deployment
